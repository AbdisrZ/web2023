# Persamaan Linear Satu Variabel

Definisi:

Persamaan linear satu variabel adalah suatu bentuk persamaan matematika yang

melibatkan variabel tunggal, biasanya dilambangkan sebagai x, dan memiliki derajat

terendah pada variabel tersebut adalah 1. Dengan kata lain, variabel x tidak memiliki

eksponen atau pangkat yang lebih tinggi dari 1.

Bentuk Umum:

Persamaan linear satu variabel dapat ditulis dalam bentuk umum ax+b=0,

dimana a dan b adalah konstanta, dan a â‰  0. Konstanta a tidak boleh sama dengan 0

karena itu akan membuat variabel x menjadi tidak relevan, dan persamaan akan

menjadi non-linear.

Contoh:

2x + 3 = 0

5x - 7 = 0

Solusi:

Solusi dari persamaan linear satu variabel ax + b = 0 dapat ditemukan dengan

mengisolasi variabel x. Solusinya adalah x = - b/a. Ini karena kita ingin mencari nilai x

yang membuat persamaan menjadi benar.

Contoh:

2x + 3 = 0

x = - 3/2

# Persamaan Linear Dua Variabel

Definisi:

Persamaan linear dua variabel adalah suatu bentuk persamaan matematika yang

melibatkan dua variabel, biasanya dilambangkan sebagai x dan y, dan memiliki

derajat terendah pada kedua variabel adalah 1. Persamaan ini dapat direpresentasikan

dalam bentuk umum ax+by=c, di mana a, b, dan c adalah konstanta, dan a dan b

tidak dapat bersama-sama menjadi 0.

Bentuk Umum:

Persamaan linear dua variabel dapat ditulis dalam bentuk umum ax+by=c, di mana:

- a, b, dan c adalah konstanta

- a dan b tidak dapat bersama-sama menjadi 0

Solusi:

Solusi dari persamaan linear dua variabel adalah pasangan nilai (x,y) yang

memenuhi persamaan tersebut. Dalam kasus sistem dua persamaan linear dua

variabel, solusinya adalah titik di mana dua garis yang dihasilkan oleh persamaan

tersebut bersilangan:

Salah satu solusinya adalah (x,y)=(2,0).
